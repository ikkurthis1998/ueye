Components

Button

Simple to use responsive buttons.

Solid Buttons

Adding btn to class list will give the base properties of a solid button. Upon adding btn-primary, btn-success and btn-error adds the respective colors shown above.

Outline Buttons

Adding btn-outline to class list on base button will add the outline styles. Upon adding btn-outline-primary, btn-outline-success and btn-outline-error adds the respective colors shown above.

Link Buttons

Adding btn-link to class list on base button will add the link styles. Upon adding btn-link-primary, btn-link-success and btn-link-error adds the respective colors shown above. Wrapping the button in an <a></a> tag will make the button a link.

Icon Buttons

Adding btn-link to class list on base button will add the link styles. Upon adding btn-link-primary, btn-link-success and btn-link-error adds the respective colors shown above. Wrapping the button in an <a></a> tag will make the button a link.
